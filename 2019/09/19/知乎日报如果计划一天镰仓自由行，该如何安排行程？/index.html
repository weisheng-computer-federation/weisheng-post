如果计划一天镰仓自由行，该如何安排行程？

图片：鸡汁的塔拉 / 知乎

鸡汁的塔拉，笑起来像个三百斤的孩子

（多图长文预警，实用攻略向）

从 2017 到 2019 年，在秋、春、夏三个季节一共去过三次镰仓，每次都能领略到镰仓和江之岛不同的美，每次都玩得心满意足！来和大家分享一下镰仓之行的攻略和心得~

先总结一下整体的攻略和推荐路线：

1，出行时间：

冬天的梅花，春天的樱花，秋天的红叶，夏天的绣球花，开阔的湘南海岸……镰仓适合一年四季来玩啦，而且离东京只要一小时电车，当天往返就足够！如果行程比较灵活的话，一定一定要挑一个大晴天来镰仓，可以远眺富士山，在江之岛看夕阳西下~

2，交通：

  • 从东京过去的话推荐从新宿出发，去小田急车站购买镰仓 / 江之岛一日周游券，包含小田急线往返镰仓的车票，镰仓江之电任意乘坐，观光景点还有门票还有优惠哦～我买了两次周游券，省事又划算~下面是小田急官网对江之岛 / 镰仓周游券的介绍：

江之岛 / 镰仓周游券 | 小田急电铁

  • 周游券可以在小田急线的旅游服务中心或者自动售票机购买。提醒大家一定要利用自动售票机！自动售票机！售票机！尤其节假日早上去新宿的服务中心买的话很容易排队（想象一下前面排了快二十个人且其中八成人买之前还要问半天路线攻略，就问你内心崩不崩溃），而旁边的自动售票机空无一人。去自动售票机买好票，到旅游中心拿一份中文纸质攻略，然后潇洒进站，上车走人，不浪费一分钟排队，简直不要太美滋滋~

3，路线：

贴一张江之岛 / 镰仓区域地图，大家也可以看 PDF 版本：镰仓 / 江之岛地图

可以看到大大小小的景点有很多，不是所有的景点都需要打卡，如果是去赏花或者看红叶的话，去主要地点就好（地图里面都有标记哦）~当然还是有些地方我非常推荐给大家打卡的！分享一下我自己玩下来总结的路线，加粗的是推荐必玩：

  • 乘坐小田急线到藤泽车站，换乘江之电到镰仓站
  • 沿小町通一路吃吃喝喝，参观鹤冈八幡宫
  • 乘坐江之电从镰仓到长谷，参观长谷寺、高德院等
  • 乘江之电到稻村崎 / 直接从长谷街沿湘南海岸步行前往镰仓海滨公园，远眺江之岛和富士山
  • 乘江之电到《灌篮高手》取景地、镰仓高校前打卡
  • 乘江之电到江之岛站，步行上岛，参观江岛神社、登江之岛展望灯台俯瞰镰仓、之后步行至江之岛西海岸，去岩洞探索，最后欣赏江之岛日落
  • 返程

（北镰仓由于位置相对偏远，考虑到之后要去江之岛没有被我包含在内，感兴趣的同学可以加入到行程哈）

———————正文分割线———————

下面和大家按区域介绍一下我在不同季节去过的镰仓~都是美好的回忆呐！

1，北镰仓：

主要景点有圆觉寺和明月院，可能位置偏僻游客相对不多，但是庭院真的充满意境～

2，镰仓站：

主要有鹤冈八幡宫和小町通，后者是镰仓美食一条街，吃了两趟的我表示一本满足！按行程的话差不多是中午饭点前后来逛，走一路吃一路，午饭都可以省啦～

江之电沿线：

中国游客最喜欢打卡的景点肯定是镰仓高校前啦，但去了三次镰仓的我只匆匆打卡了一次～湘南海岸的美可不止于此。

去看海之前先坐江之电到长谷站下车，附近有很多庭院和寺庙，不同季节也会看到应季的花朵盛开～

相比镰仓高校前，我个人更想给大家推荐的地方是镰仓海滨公园，可以坐江之电到稻村崎然后步行前往，也可以从长谷街走到海边然后沿着海一路走过去，感受汹涌的浪花拍击海岸，看冲浪的人们玩得好不快活~

从山上的展望台一路看过去，湘南海岸线，江之岛，远处的富士山尽收眼底。

去江之岛之前可以顺路打卡一下镰仓高校车站，像我这种没什么《灌篮高手》情结的只去过一次，不走心地拍张照就结束了哈哈。

江之岛：

一个步行即可游览完，内容却非常丰富的小岛～建议下午三四点上岛，游览完景点可以看夕阳哦。上山时有登山电梯可以坐，但我觉得自己走才能看到更多好风景～

走到岛上西边的海岸，如果离日落还有段时间，可以先去由波浪侵蚀而形成的江之岛岩屋探索一番～

镰仓之行最美的结尾就是坐在海边看夕阳啦，如果和心爱的人一起看就更浪漫啦^^

夜幕降临，恋恋不舍地告别江之岛，告别富士山，告别湘南海岸，镰仓一日行完美收官。

希望大家也都能在镰仓和江之岛留下美好的回忆~
